# Global Income Analysis Dashboard

An interactive data visualization platform that makes complex economic data accessible to everyone. Explore global income distribution, inequality patterns, and economic disparities across countries and regions.

## 🌍 Overview

The Global Income Analysis Dashboard combines datasets from international sources and presents them through clear visual analytics. Our goal is to make economic data accessible so that anyone — from students to policymakers — can understand how wealth and opportunity are distributed around the world.

## ✨ Features

- **Global Income Comparison**: Compare income levels across 54 countries and 6 regions
- **Inequality Indicators**: Analyze Gini Index, GDP per capita, and income distribution
- **Interactive Data Visualization**: Filter and explore trends across 20+ years of data
- **Power BI Integration**: Advanced analytics with React-based navigation
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## 🛠️ Tech Stack

- **Frontend**: React 18, React Router
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Analytics**: Power BI Embedded
- **Deployment**: Modern web standards

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Shravani-git/global-income-inequality-dashboard.git
cd global-income-dashboard
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

### Build for Production

```bash
npm run build
npm run preview
```

## 🎯 Key Metrics

- **GDP**: Gross Domestic Product
- **Gini Index**: Income inequality measurement (0-100)
- **GDP per Capita**: Average economic output per person
- **Income Distribution**: Wealth distribution across population segments
